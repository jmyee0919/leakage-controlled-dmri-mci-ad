# Data directory

This repository does not include ADNI participant-level data.

Authorized ADNI users should place required raw files in `data/raw/` and write derived intermediate files to `data/processed/`.

Do not commit participant-level ADNI files, subject-level derived CSVs, prediction files, visit-level files, DICOM, NIfTI, FreeSurfer outputs, or files containing RID/PTID/SubjectID/date-level information.
