# Excluded files manifest

The original working archive contained analysis intermediates, manuscripts, figures, ADNI-derived subject-level tables, and exploratory scripts. The cleaned repository excludes restricted or non-public materials, including:

- ADNI-derived participant-level or visit-level CSVs
- filtered ADNI snapshot/DeltaMRI tables
- subject-level prediction and out-of-fold files
- raw neuroimaging files
- large exploratory archives
- manuscript Word/PDF drafts not needed for code reproduction

The public repository contains only curated source code, aggregate results, figures, configuration files, and documentation.
